# UI 组件库

## 1. 轮播图组件
  1. autoPlay="true" 自动轮播配置
  2. autoPlayDirection="next" 自动轮播方向
  3. durations="3000" 轮播一次时间
  4. initial="1" 初始图片
  5. hasDot="true" 是否显示指示原点
  6. hasDirector="true" 显示方向按键
  7. dotBgColor="hotpink" 圆点背景颜色
